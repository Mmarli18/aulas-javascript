Exercício

Crie um arquivo chamado exercicio02.html. A partir dele, faça um script chamado exercicio02.js (não se esqueça de usar a pasta js).

Crie um array de Clientes contendo 3 objetos. Cada objeto deve ter um identificador (1, 2 e 3) e o nome do cliente (Goku, Naruto, Shiryu).

Faça um loop (qualquer um dos que vimos) e mostre no console os dados de Cliente conforme a seguir:

Cliente: Goku (id: 1) Cliente: Naruto (id: 2) Cliente: Shiryu (id: 3)
